There are 2 parts to this lab: 
1. A rules-based supervisor/worker structure. This is a well established pattern. Though it's not agentic, it is a classic supervisor/worker relationship.
2. An Agents-based supervisor/worker structure. To run this lab, you need an [OpenAI API key](https://platform.openai.com/api-keys). I believe you need to load some amount of credits. If you do not want to do this, just read through the lab to understand how the supervisor agent orchestrates the delegation of tasks.
